## Overview
The Surface module allows admins to **control the visibility of UI containers** on a per-user or global basis.

## Current Features

- Show **global** (all users) and **user-specific** rules.
- Inline deletion of individual rules via the modal.
- Rule creation via container hover interface.
- Display creator and creation date for audit purposes.
